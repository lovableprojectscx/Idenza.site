## Arquitectos de Experiencias Digitales

Elevamos estándares. Fusionamos ingeniería de software con estética de vanguardia para crear productos digitales que no solo funcionan: inspiran.

## Technologies used

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Framer Motion

## Getting Started

To run the project locally:

1. Clone the repository
2. Install dependencies:
   ```sh
   npm install
   ```
3. Start the development server:
   ```sh
   npm run dev
   ```
